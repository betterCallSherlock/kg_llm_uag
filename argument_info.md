| 参数            | 说明                 | 默认值                        |
| ------------- | ------------------ | -------------------------- |
| `--train`     | 训练集路径              | `./data/WebQSP_train.json` |
| `--test`      | 测试集路径              | `./data/WebQSP_test.json`  |
| `--max_train` | 最大训练样本数            | 全部                         |
| `--max_test`  | 最大测试样本数            | 全部                         |
| `--alpha`     | 目标误差率（1−α为目标置信覆盖率） | `0.2`                      |
| `--max_hop`   | 最大推理跳数             | `2`                        |
| `--out_json`  | 输出结果保存路径           | 无                          |
